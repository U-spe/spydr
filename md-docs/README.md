## Overview

spydr is a lightweight web interface designed to feel more like an operating system than a traditional website.

Built with speed, simplicity, and clean visuals in mind, it provides a centralized hub for launching games, web apps, AI tools, and a built in web browser, all while maintaining a minimal grayscale aesthetic.

---

## Features

- Fast and lightweight
- Modern grayscale UI
- Responsive design
- Built-in game launcher
- Built-in web browser
- App launcher
- AI integration
- Keyboard shortcuts
- Tab cloaking
- Custom themes
- Animated background effects
- Modular architecture
- JSON-powered content loading

## Getting Started

Clone the repository.

```bash
git clone https://github.com/U-spe/spydr.git
```

Enter the project.

```bash
cd spydr
```

If you're using a local server:

```bash
npm install
npm start
```

Or simply serve the project using any static web server.

---

## Configuration

Most site content is loaded through JSON files.

Example:

```json
{
  "title": "Minecraft",
  "url": "/games/minecraft/",
  "image": "/assets/images/minecraft.webp"
}
```
