# 🕷️ SPYDR

> a minimal, grayscale web hub built for speed, clarity, and control.

---

## ⚙️ overview

SPYDR is a lightweight web interface designed like a system dashboard — clean UI, fast navigation, and a cinematic boot experience.

it’s built to feel less like a website and more like a live environment.

---

## 🎨 design system

### colors

- background: `#090909`
- secondary: `#111111`
- cards: `#171717`
- borders: `#2b2b2b`
- white: `#ffffff`
- gray: `#bdbdbd`

### accent (minimal use)

- deep blue: `#3f5efb`
- purple: `#7c3aed`

---

### typography

- headers: Space Grotesk
- UI / body: JetBrains Mono

---

## 🧠 features

- cinematic loading screen (video boot sequence)
- system clock + date display
- hover-based navigation menu
- grayscale UI system (clean + minimal)
- mini game card launcher
- smooth transitions across all UI elements
- scalable structure for apps, games, and tools

---

## 🕸️ philosophy

SPYDR is built around one idea:

> less interface, more system.

no clutter. no noise. no unnecessary effects.

just a controlled environment that feels like a live OS layer in the browser.
